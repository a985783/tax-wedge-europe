# 替代解释排除方案 (Alternative Explanations)

在建立 VAT 传导机制时，必须排除以下竞争性解释：

## 1. 内生性政策变动 (Endogenous Policy Changes)
*   **解释**: 政府可能在通胀预期高企时提高 VAT，或者在通缩时降低 VAT。观察到的价格变化可能反映了宏观趋势而非税收传导。
*   **排除策略 (Exclusion Strategy)**:
    *   **平行趋势检验 (Parallel Trends Test)**: 检查 $t < 0$ 期的系数 $\beta_{pre}$ 是否显著异于 0。
    *   **结果**: Event study 图显示 $t < 0$ 区间系数平坦且接近 0，支持外生性假设。
    *   **控制变量**: 在更严格的模型中，可控制 GDP 增长率或产出缺口（当前模型通过 `rel_time` 固定效应和堆叠法部分吸收了平均趋势，但未来可加强）。

## 2. 季节性效应 (Seasonality)
*   **解释**: 某些商品（如食品、旅游服务）价格具有强季节性。如果税收调整往往发生在特定月份（如1月），结果可能混杂季节性因素。
*   **排除策略**:
    *   **归一化方法**: 使用相对于 $t=-1$ 的变化，并控制 `C(rel_time)`。
    *   **多国多时点**: 利用 Panel 数据中事件发生的交错时间（Staggered timing）来抵消特定月份的季节性。
    *   **Robustness Check**: 可以尝试去季节性后的 HICP 数据（HICP_SA）进行复现（如有）。

## 3. 商业周期效应 (Business Cycle Effects)
*   **解释**: 经济衰退期间的需求弹性可能不同，导致传导率差异。
*   **排除策略**:
    *   **异质性分析**: 检查不同时间段（如危机前 vs 危机后）的传导率。
    *   **当前处理**: 样本覆盖多年，平滑了单一周期的影响。

## 4. 其它成本冲击 (Other Cost Shocks)
*   **解释**: 油价冲击或汇率波动可能与 VAT 调整同时发生。
*   **排除策略**:
    *   **大样本优势**: 使用大量不同国家、不同商品的事件，特异性的成本冲击应当相互抵消（除非是全球性同步冲击且与 VAT 政策高度相关，这不太可能）。
    *   **控制组**: 理想情况下使用未受冲击的类似商品作为对照组（Difference-in-Differences）。当前的 Event Study 本质上是利用自身时间序列变化，未来可引入 Control Group（Clean Control）。

## 5. 测量误差 (Measurement Error)
*   **解释**: 税收变化幅度 `delta_tw` 可能测量不准。
*   **排除策略**:
    *   **样本筛选**: 只保留 `|delta_tw| > 1%` 的显著变动事件，降低微小噪音的影响。
    *   **加权回归**: 使用 CPI 权重进行加权回归，确保结果代表宏观价格水平。
