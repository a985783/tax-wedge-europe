## 提交SSRN前的建议

1. **作者信息**：请在标题页填入实际作者姓名、单位和邮箱
2. **ORCID**：建议在作者信息后添加ORCID编号
3. **致谢**：如有致谢，请在引言前添加
4. **资助声明**：如有资助，请在引言前添加

## 如何提交到SSRN

1. 访问 https://www.ssrn.com/
2. 创建/登录账户
3. 选择"Submit a Paper"
4. 上传 `paper.pdf`
5. 填写元数据（标题、作者、摘要、关键词、JEL分类码）
6. 提交
